# 鸢尾花数据集 3D 可视化实验

## 项目描述
本项目基于鸢尾花数据集，使用 Python 和逻辑回归模型，实现了两种 3D 可视化任务：
1. **任务二：3D Decision Boundary（决策边界可视化）**
2. **任务三：3D Probability Map（概率分布可视化）**

通过本实验，可以直观理解逻辑回归模型在三维特征空间中的分类行为和概率分布情况。

---

## 实验目的
- 展示逻辑回归模型在三维特征空间中的决策边界
- 展示模型在整个特征空间中的概率分布
- 理解线性分类器在三维空间中的划分方式
- 分析特征对分类结果和置信度的影响

---

## 数据集
- 使用鸢尾花数据集的前三个特征：
  - Sepal Length（花萼长度）
  - Sepal Width（花萼宽度）
  - Petal Length（花瓣长度）
- 选取前两个类别（Setosa 和 Versicolor），实现二分类任务

---

## 模型与参数
- 模型：逻辑回归（Logistic Regression）
- 训练集：70%
- 测试集：30%
- 模型参数：
  - \( w_1 = 0.512 \)
  - \( w_2 = -1.253 \)
  - \( w_3 = 2.678 \)
  - \( b = -0.891 \)
- 测试集准确率：96.67%

---

## 可视化结果
实验生成了两个 3D 可视化图形，合并展示如下：

### 1. 任务二：3D Decision Boundary
- 灰色半透明平面表示决策边界
- 红色点为 Setosa 类别
- 蓝色点为 Versicolor 类别
- 平面方程为：
  \[
  0.512x_1 - 1.253x_2 + 2.678x_3 - 0.891 = 0
  \]

### 2. 任务三：3D Probability Map
- 彩色曲面表示属于类别 1 的预测概率
- 颜色从蓝色（0）到红色（1）
- 曲面高度直接对应概率值
- 颜色条标注概率与颜色对应关系
